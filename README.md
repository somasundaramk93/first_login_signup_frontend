# Login & Signup Frontend Project

A simple React frontend application for handling user login and signup forms with validation.

---

## Features
- User-friendly login and signup forms
- Form validation (email format, password length)
- Responsive design for desktop & mobile
- Navigation between login and signup pages using React Router
- Optional integration with backend API for authentication

---

## Tech Stack
- React.js
- HTML5, CSS3, JavaScript (ES6+)
- React Router
- Axios / Fetch API (optional for API integration)
